# Zerodha Clone – Stock Trading Platform  

A full-stack project inspired by Zerodha, India’s leading stock brokerage platform.  
This project demonstrates frontend design, backend development, and stock trading simulation (without live market data).  

## 🚀 Features
- **Frontend Clone** of Zerodha’s official website (landing page + UI)  
- **Trading Dashboard** to view portfolio and execute buy/sell trades  
- **User Authentication** (signup/login)  
- **Static Stock Data Simulation** 
 

## 🛠️ Tech Stack
- **Frontend:** React (HTML, CSS, JavaScript,Bootstrap)  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Version Control:** Git, GitHub  



## 📌 How to Run
1. Clone the repository:

   git clone https://github.com/ADITYALAVAND/Stock-Trading-Platform-.git

## Install dependencies for both frontend and backend:

  cd frontend && npm install
  cd ../backend && npm install

## Run the backend,frontend,dashboard:

  npm start
